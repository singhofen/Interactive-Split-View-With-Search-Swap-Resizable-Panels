# Interactive-Split-View-With-Search-Swap-Resizable-Panels
Interactive Bible Split-View with all 66 books (OT &amp; NT). Features: - Draggable resizable divider - Swap panels button (keeps independent scroll) - Search bar on both sides - Dark glass UI (Glassmorphism) - Fully scrollable, responsive, and modern design  Perfect for Bible study, web UI demos, or portfolio showcase.
