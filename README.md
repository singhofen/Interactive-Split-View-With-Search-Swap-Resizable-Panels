# Bible Split-View Web App

## Overview
Interactive Bible Split-View featuring **all 66 books** of the Bible (Old Testament and New Testament). Users can:

- Scroll each panel independently
- Swap left/right panels while preserving scroll position
- Resize panels with a draggable divider
- Search books and summaries on each side
- Enjoy a modern **dark glass UI (Glassmorphism)**

Perfect for **Bible study, web UI demos, or portfolio showcases**.

## Features
- **Interactive split panels**: OT on left, NT on right  
- **Resizable divider**: Drag to adjust panel width  
- **Swap button**: Switch left/right panels seamlessly  
- **Independent scroll**: Each panel scrolls separately  
- **Search bars**: Filter books and summaries in real-time  
- **Dark Glass UI**: Modern, responsive, and visually appealing  
- **Mobile-friendly**: Works on desktop and mobile devices  

---

## Tech Stack
- **HTML5**  
- **CSS3** (Glassmorphism styling)  
- **Vanilla JavaScript** (no frameworks)  
- Fully responsive and interactive  

---

## Installation / Usage
1. Copy the HTML, CSS, and JS code into your own project or CodePen.  
2. Open in any modern browser.  
3. Interact with the panels: resize, scroll, swap, and search.  

No backend or dependencies are required.
